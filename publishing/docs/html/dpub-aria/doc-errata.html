Description

A set of corrections discovered after initial publication of the work, sometimes referred to as corrigenda.

Examples

Relation to ARIA

doc-errata is a landmark role.

Usage

doc-errata is typically used on section elements. Do not use the role on list elements. List(s) of errata occurs inside the element carrying the doc-errata role.

A label must be provided for the role using one of the aria-labelledby, aria-label or title attributes.

For a simplified list of all the elements the role is allowed on, refer to the mapping table in the EPUB Type to ARIA Role Authoring Guide. For the official list of allowed elements, refer to ARIA in HTML.
