Description

An explanatory or alternate title for the work, or a section or component within it.

Examples

Relation to ARIA

doc-subtitle inherits its semantics from the a sectionhead role.

Usage

doc-subtitle is typically used on p and div elements. For more information on authoring subtitles, see the section Subheadings, subtitles, alternative titles and taglines in the HTML 5.1 specification.
