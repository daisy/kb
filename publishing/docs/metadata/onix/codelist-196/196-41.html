Explanation

Code 41 indicates that a publication includes page number navigation (e.g., a page list).

It is not required that there be a pre-paginated source, such as a source print book. Page navigation may be included for digital-only publications.

Refer to the Page List Navigation section of the knowledge base for more information.

See also code 19 when including page break markers.

Only for use in ONIX 3.0 or later.
