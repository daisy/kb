Explanation

Code 28 is used to indicate that all or substantially all non-text content has full alternative descriptions as pre-recorded audio.

This code applies if extended descriptions, transcripts, etc. also offer an option to be listened to as pre-recorded audio.

For video-only content, audio descriptions must be provided.

Only for use in ONIX 3.0 or later.
