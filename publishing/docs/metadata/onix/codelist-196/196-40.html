Explanation

Code 40 indicates that the purpose of each link, control, or button in a publication is clearly understandable either from its text content alone or from an associated description.

This code overlaps with WCAG 2 success criteria 2.4.4 Link Purpose (In Context) and 2.5.3 Label in Name.

Only for use in ONIX 3.0 or later.
