Explanation

Code 29 is used to indicate that the markup accurately represents the structure and headings of the content, thereby allowing users to navigate the content using assistive technology shortcuts.

This code is generally the equivalent of meeting the structuring requirements of Success Criterion 1.3.1 Info and Relationships.

Only for use in ONIX 3.0 or later.
