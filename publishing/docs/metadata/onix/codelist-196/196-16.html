Explanation

Code 16 is used to indicate that there are data visualizations in the publication and that the data they contain is also provided in textual form. The code overlaps with with code 15 but indicates this more specific type of content is present and has extended descriptions.

For graphs, charts, and other types of diagrams, including a table of equivalent data or a description of the data meets the requirement.
