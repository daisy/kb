Explanation

Code 84 is used to indicate that an EPUB publication that conforms to the EPUB Accessibility standard (version 1.1 or higher) conforms at Level A of the specified version of WCAG 2.

This code must be paired with code 04 as well as one of the WCAG version indicators (code 80, 81, or 82).

Only for use in ONIX 3.0 or later.
