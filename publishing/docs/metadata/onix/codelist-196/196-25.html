Explanation

Code 25 indicates that a publication is optimized so that color is not the sole means of conveying information.

This code corresponds to WCAG Success Criterion 1.4.1 - Use of Color. Publications should pass the requirements of that success criterion in order to declare this code.

Note that this code is for visual reading. The methods for passing it do not ensure that non-visual readers can access the same information.

Only for use in ONIX 3.0 or later.
