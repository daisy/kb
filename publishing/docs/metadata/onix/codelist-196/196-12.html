Explanation

Code 12 indicates that a digital publication includes a working index — one that either hyperlinks users to the location where terms are discussed or provides a similar access mechanism (e.g., includes a page list so that users can jump to the static page markers in their reading system.

A publication with an index but no means of reaching the referenced locations does not meet the requirements of this code.

Refer to the Indexes page in the knowledge base for more information on tagging indexes.
