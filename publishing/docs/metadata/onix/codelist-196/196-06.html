Explanation

Code 06 is used to indicate that a PDF conforms to the requirements of ISO 32000-2:2020 and ISO 14289-2:2014 (PDF/UA-2).

Only for use in ONIX 3.0 or later.

Note

As the DAISY Knowledge Base does not address PDF accessibility, please refer to any applicable ONIX documentation for more information on using this code.
