Explanation

Code 25 is used to indicate that the publisher has not been able to determine if the content presents a sound hazard.

As sound hazards are currently not well defined, using this value when a publication contains sounds is common. It is not necessary to detail why the presence of sound hazards is unknown.

Only for use in ONIX 3.0 or later.
