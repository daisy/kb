Explanation

Code 18 is used to indicate that a publication does not contain any known motion simulation hazards.

If there are motion simulation hazards, use code 17. If the risk cannot be determined, use code 26.

When using this code, make sure to also indicate if the publication has flashing or sound hazards.

If a publication does not contain any hazards, use code 00 instead of setting each individual code.

Only for use in ONIX 3.0 or later.
