Explanation

Code 26 is used to indicate that the publisher has not been able to determine if the content presents motion simulation hazards.

When indicating that a motion simulation hazard is not known, the reason why it is not known should be detailed in the accessibility addendum.

Only for use in ONIX 3.0 or later.
