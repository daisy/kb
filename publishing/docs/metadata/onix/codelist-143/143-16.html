Explanation

Code 16 indicates that a publication does not contain any known sound hazards.

If there are sound hazards, use code 15. If the risk cannot be determined, use code 25.

When using this code, make sure to also indicate if the publication has flashing or motion simulation hazards.

If a publication does not contain any hazards, code 00 can alternatively be set instead of each individual code.

Only for use in ONIX 3.0 or later.
