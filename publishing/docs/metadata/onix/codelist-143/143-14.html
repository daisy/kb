Explanation

Code 14 indicates that a publication does not contain any known flashing hazards.

If there are flashing hazards, use code 13. If the risk cannot be determined, use code 24.

When using this code, make sure to also indicate if the publication has sound or motion simulation hazards.

If a publication does not contain any hazards, code 00 can alternatively be set instead of each individual code.

Only for use in ONIX 3.0 or later.
