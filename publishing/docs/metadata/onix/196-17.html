Explanation

Code 17 is used to indicate that MathML markup has been provided for math content.

Although the ONIX description for this code prefers the inclusion of Semantic MathML, web user agents, such as EPUB reading systems, only support the display and voicing of Presentational MathML. It is generally unrealistic to include Semantic MathML.
