Explanation

Code 81 is used to indicate that an EPUB publication that conforms to the EPUB Accessibility standard (version 1.1 or higher) conforms to WCAG 2.1.

This code must be paired with code 04 as well as one of the WCAG level indicators (code 84, 85, or 86).

Only for use in ONIX 3.0 or later.
