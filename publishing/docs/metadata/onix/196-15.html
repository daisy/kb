Explanation

Code 15 is used to indicate that all non-text content that cannot be explained through alternative text alone (see code 14) include extended descriptions.

For images, this means that all complex images include extended descriptions.

For audio content, meeting this requirement can be achieved through the provision of transcripts or captions.

For video-only content, a description of what is occurring in the video would suffice.

It is not required to provide extended descriptions for content that conveys no meaningful information, such as decorative images.

Refer to the Image Descriptions, Audio, and Video sections of this site for more information on meeting this requirement.
