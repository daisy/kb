Explanation

Code 38 is used to indicate that definitions are provided for any unusual words, idioms, jargons, or abbreviations.

Definitions could be provided inline, in the case of expanded abbrevations, or could be contained in a glossary, either per chapter or at the end of the work.

Refer to the Glossaries page in the knowledge base for more information on tagging indexes.

Only for use in ONIX 3.0 or later.
