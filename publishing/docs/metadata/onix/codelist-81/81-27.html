Explanation

Code 27 is used to indicate that a publication contains video of visual performances, such as clips of a play, dance, or musical performance. It is the equivalent of setting a visual access mode using schema.org metadata.

This code is typically set in a <ProductContentType> element as video clips of performance are usually supplemental content.

Only for use in ONIX 3.0 or later.
