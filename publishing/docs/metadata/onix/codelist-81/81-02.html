Explanation

Code 02 indicates that the publication contains a spoken word performance, such as a play.

The code is set in a <PrimaryContentType> element.

Only for use in ONIX 3.0 or later.

Schema.org mapping

Code 02 is the equivalent of setting an auditory access mode using schema.org metadata.
