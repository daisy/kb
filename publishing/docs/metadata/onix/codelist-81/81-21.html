Explanation

Code 21 indicates that the publication contains a partial spoken word recording of the work.

Code 21 could be set in either a <PrimaryContentType> element if the audio performance is all the publication contains (i.e., it is an audiobook), or in a <ProductContentType> element if the performance is included with a textual or visual publication.

Only for use in ONIX 3.0 or later.

Schema.org mapping

Code 21 is the equivalent of setting an auditory access mode using schema.org metadata.
