Explanation

Code 26 is used to indicate that a publication contains a video recording of a reading of itself or another work. It is the equivalent of setting a auditory access mode using schema.org metadata.

The code is set in a <ProductContentType> element as video clips of readings are supplemental content.

Only for use in ONIX 3.0 or later.
