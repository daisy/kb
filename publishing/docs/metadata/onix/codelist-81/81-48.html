Explanation

Code 48 is used to indicate that a publication contains math content. Depending on how the math is encoded it is the equivalent of setting the mathOnVisual access mode (for images of math) or the textual access mode (for MathML markup, latex, etc.) using schema.org metadata.

Code 48 is typically set in a <ProductContentType> element for digital publications. Math equations and formulae are typically only part of the text content.

Only for use in ONIX 3.0 or later.
