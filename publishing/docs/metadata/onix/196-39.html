Explanation

Code 39 is used to indicate that all supplemental textual and visual material necessary to understand the publication is either available as text for text-to-speech rendering or pre-recorded audio is provided for it.

Supplemental materials could include sidebars, glossaries, etc.

See also code 20, which covers synchronized pre-recorded audio, and code 28, which covers audio for non-text descriptions.

Only for use in ONIX 3.0 or later.
