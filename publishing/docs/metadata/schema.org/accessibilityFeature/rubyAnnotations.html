Definition

Indicates that ruby annotations are attached to some but not all CJK ideographic characters in the content.

Examples

Explanation

The rubyAnnotations value is used to indicate that a publication includes ruby annotations.

Ruby annotations are most commonly used to provide pronunciation guides for Chinese, Japanese and Korean texts.

ONIX Mapping

There is currently no equivalent ONIX metadata code for this value.
