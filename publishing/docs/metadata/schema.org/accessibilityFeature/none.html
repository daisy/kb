Definition

Indicates that the resource does not contain any accessibility features.

Examples

Explanation

The none value indicates that a publication does not include any accessibility features.

This value exists so that there is no ambiguity about whether a publication does not contain any accessibility features or whether it was never checked for them.

ONIX Mapping

The none value maps to list 196 code 09 ("Inaccessible, or known limited accessibility").
