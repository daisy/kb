Version: WCAG 2.0 and up
Number: 2.4.5
Level: AA
Applicability:
  • Index
  • Logical Reading Order
  • Page List
  • Table of Contents
Requirement:

More than one way is available to locate a Web page within a set of Web pages except where the Web Page is the result of, or a step in, a process.
Guidance:
  • Understanding Success Criterion 2.4.5
  • How to meet Success Criterion 2.4.5
Techniques:

  • G64: Providing a Table of Contents
  • G125: Providing links to navigate to related Web pages
  • G126: Providing a list of links to all other Web pages
  • G161: Providing a search function to help users find content
  • G185: Linking to all of the pages on the site from the home page

Purpose

The purpose of this success criterion is to help ease navigation of a digital publication.

How to Meet

This success criterion is typically met by all EPUB publications provided they include a table of contents and all documents are listed in the spine.

Other means of navigation that authors can include for users include page lists, indexes, and lists of illustrations, tables, and examples.
