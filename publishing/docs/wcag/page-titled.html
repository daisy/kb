Number: 2.4.2
Level: A
Applicability:
  • Document Title
Requirement:

Web pages have titles that describe topic or purpose.
Guidance:
  • Understanding Success Criterion 2.4.2
  • How to meet Success Criterion 2.4.2
Techniques:
  • G88: Providing descriptive titles for Web pages
  • H25: Providing a title using the title element

Purpose

The purpose of this success criterion is to ensure that users can easily orient themselves to their location within a digital publication (e.g., when following a link from an index).

How to Meet

Authors need to ensure that each document in the publication has a title element that describes the content of the document.
