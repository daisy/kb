Version: WCAG 2.0 and up
Number: 3.3.5
Level: AAA
Applicability:
  • Form Inputs
Requirement:

Context-sensitive help is available.
Guidance:
  • Understanding Success Criterion 3.3.5
  • How to meet Success Criterion 3.3.5
Techniques:
  • G71: Providing a help link on every Web page
  • G89: Providing expected data format and example
  • G184: Providing text instructions at the beginning of a form or set of fields that describes the necessary input
  • G193: Providing help by an assistant in the Web page
  • G194: Providing spell checking and suggestions for text input

Purpose

The purpose of this success criterion is to help users complete forms by providing instructions that explain the purpose and expected input of complex fields.

How to Meet

For input fields that are not easily understood from their name alone, authors also provide the option for users to access context-sensitive help instructions.

The instructions might provide an example of the expected input or explain in more detail the kinds of information the user is expected to input.
