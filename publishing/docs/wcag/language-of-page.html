Version: WCAG 2.0+
Number: 3.1.1
Level: A
Applicability:
  • Text
Requirement:

The default human language of each Web page can be programmatically determined.
Guidance:
  • Understanding Success Criterion 3.1.1
  • How to meet Success Criterion 3.1.1

Purpose

The purpose of this success criterion is to ensure that the language of the content is available to reading systems and assistive technologies so that they can best render the text content (e.g., to optimize text-to-speech playback).

How to Meet

Authors need to ensure that the primary language is specified both in the EPUB Package Document metadata (using xml:lang attributes and the dc:language element) and in any HTML and SVG documents with text content (using the xml:lang and/or lang attributes, as appropriate).
