This page is continuously updated to reflect significant changes to the DAISY Accessible Publishing Knowledge Base.

To closely follow changes to this site, it is recommended to watch the project's GitHub repository. If you are not interested in all the discussions that go on, you can select to watch only pull requests, as these contain the changes to the site.

Changes are listed from most recent to oldest.

August 28, 2025 — Accessibility hazards updates

The accessibility hazard values are now covered in their own pages, providing updated information, examples of each value, and mappings to their ONIX equivalents.

The coverage of values has also been updated to include the individual unknown hazard properties - unknownFlashingHazard, unknownMotionSimulationHazard, and unknownSoundHazard.

August 26, 2025 — Accessibility feature updates

New pages are now available describing how to use the recently added latex-chemistry and MathML-chemistry pages.

The guidance on using the annotations value has been removed as the value has been deprecated.

August 26, 2025 — Added mappings from acceessibility features to ONIX

The accessibilityFeature value pages now include mappings to their ONIX counterparts.

August 25, 2025 — Added coverage of ONIX codelist 81

Codelist 81 describes the type of content in a publication. It is equivalent to the schema.org access modes.

August 12, 2025 — Added coverage of ONIX codelist 143

The section on ONIX metadata has been expanded to include coverage of codelist 143. This list defines codes for expressing information about potential hazards.

August 8, 2025 — ONIX metadata section expanded

The ONIX metadata page has been replaced by new pages that individually cover each of the codes in list 196. The update also adds coverage of all the new codes added in recent ONIX updates. Future updates will add coverage of codes outside of list 196 that are used to express accessibility information.

July 8, 2025 — Prioritizing links to image descriptions

The page on providing image descriptions has been updated to prioritize the current best practice of using the aria-details attribute to reference a hyperlink to the description.

April 2, 2025 — Knowledge base now available in Japanese

The knowledge base has been translated into Japanese. It is also available to Japanese users in the Ace by DAISY validation tool.
