「アクセシブル出版 ナレッジベース」は、もともと 2012 年に出版された O'Reillyアクセシブル EPUB 3電子書籍を補完するものとして開発されました。書籍の執筆の一環として、収集された情報を確認するために作成されましたが、すべての情報が書籍に掲載されたわけではありません。

このサイトの最初のバージョンはDAISYによって開発され、（現在は廃止されている）International Digital Publishing Forum (IDPF) の Web サイトでホストされていました。IDPF は元々EPUB規格の作成とメンテナンスを担当していた組織です。ハイブリッド ガイドラインもEPUB出版物としてパッケージ化されていました。

IDPF が World Wide Web Consortium (W3C) と合併したとき、DAISYがサイトのホスティングを引き継ぎ、コンテンツの管理者としての役割を継続しました。ナレッジベースはそれ以来成長を続け、アクセシブルな電子出版物の作成に関する幅広いトピックを網羅するようになりました。このサイトは依然として EPUB 3 出版物のアクセシブル化を主な目的としていますが、EPUB 2 のアクセシブルな実践例の情報源でもあり、 W3C オーディオブックでアクセシブルな HTML コンテンツを作成するための実用的なガイドラインも提供しています。

その後、EPUB 3 出版物のアクセシビリティをチェックするための正式なフレームワークを定義するEPUB アクセシビリティ規格も公開されました。このサイトは現在、この規格を満たすコンテンツを作成するための主要な情報源となっています。
