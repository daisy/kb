Summary

Ensuring all identifiers are unique prevents problems navigating and accessing the content.

Techniques

  • Verify that all id attributes have unique values. (WCAG 4.1.1)
    • H93: Ensuring that id attributes are unique on a Web page

Examples

Explanation

If all the id attributes in a document are not unique, a reading system will not understand which one is being referred to. For example, if a user is trying to navigate to a specific section, the reading system will not be able to determine which of the sections to go to. It might take the user to the first section with the identifier, or it might leave the user at the top of document.

Duplicate identifiers are an XML error, so also be aware that they can cause a reading system to report an error instead of rendering the document. In this case, the user won't have any access to the content.
